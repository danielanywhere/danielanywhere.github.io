ContainerBase

Base class used to create full container items.

This class doesn't include the static Parse method with the signature: public static int Parse(XInfoCollection values, TextLineCollection lines, int index, int indent, CommentInfoCollection comments)

or the static Pattern property with the signature: public static string Pattern { get { return mPattern; } }

Parse method and Pattern property must both be implemented by you.
