      SvgToolsLib Namespace

      An easy-to-implement function library that performs commonly-needed tasks related to SVG files, like those exported from Inkscape, Adobe Illustrator, Canva, Figma, or others. See the full project source at https://github.com/danielanywhere/SvgTools
